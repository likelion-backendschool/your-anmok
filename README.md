## 프로젝트 주요 기능

---

> 카테고리에 어울리는 추천 장소를 확인하고 등록할 수 있어요

> 나만의 카테고리를 생성할 수 있어요

> 카테고리 홈으로 들어가 카테고리를 한눈에 확인할 수 있어요

> 장소에서 번개모임을 등록할 수 있어요

> 번개 모임

> 마이페이지에서 닉네임, 북마크, 번개 모임, 댓글을 관리할 수 있어요
